# Automation Scripts

## update_readme.py

Updates the organization README based on EST time. Runs automatically every 3 hours via GitHub Actions.

**Usage:**
```bash
pip install -r requirements.txt
python update_readme.py
```

**Time Themes:**
- 3am-6am: Night Watch
- 6am-9am: Dawn Brief  
- 9am-12pm: Session Active
- 12pm-3pm: Peak Analysis
- 3pm-6pm: Action Hours
- 6pm-9pm: Evening Brief
- 9pm-12am: Night Watch
- 12am-3am: Deep Watch
